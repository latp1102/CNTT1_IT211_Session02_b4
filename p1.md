So sánh SOAP và REST
SOAP
Là giao thức Web Service chuẩn
Sử dụng XML
Hỗ trợ bảo mật nâng cao
Hỗ trợ giao dịch phân tán và tính toàn vẹn dữ liệu cao
Phức tạp nhưng rất ổn định

REST
Là kiến trúc API dựa trên HTTP
Thường dùng JSON → nhẹ và nhanh
Dễ phát triển, dễ mở rộng
Phù hợp hệ thống hiện đại và microservices
